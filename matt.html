Personal projects relevant to IOT/monitoring:

Utility trailer temp monitor.
Using an ESP8266, DHT22 temp sensor, and a couple relays, I built a temperature monitor for a utility trailer that needed to stay above 10°C. In addition to transmitting temperature and humidity data to adafruit io for display on a dashboard, the system drives red and green indicator lights mounted outside the trailer to provide fast and intuitive visual indication of temperature status. ESP code and relay logic is set up to indicate red in the event of a failure of any component in the sensor system.
Well flow tester and data logger.
My dad, who's a home inspector, needed a device to test and log water flow from residential wells. There are commercial solutions, but they're far too expensive for this application. As a rapid proof of concept, I created a logging device using an inexpensive water meter, LogAlert for data acquisition, rechargable USB power, and a toolbox with 3d printed mounts.
Utimately I would like to automate the report generation using Python. But for now, the data is exported from LogAlert and dumped into an Excel template which auto-adjusts for test duration and presents the data in a usable format for customers. Pump current logging is another feature that is currently manually acquired and entered but I would like to automate this as well with a CT that the LogAlert can read and log.
